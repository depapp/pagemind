üí° What's Happening Behind the Scenes:

  • When you click a history item, the extension stores the summary data temporarily

  • When you open the extension in that tab, it checks if the URL matches any pending summary
  • If it matches, it displays the cached summary automatically
  • The cache indicator shows that it came from Redis cache
